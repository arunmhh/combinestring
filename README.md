
# stringCombineAddin

**Quickly convert selected text into an R character vector with double quotes.**

This R package provides an **RStudio addin** and a function to turn your selected text into a quoted R vector, making it easy to prepare lists for analysis.

---

## **Installation**

Install the package directly from GitHub:

```r
# install remotes if not already installed
install.packages("remotes")

# install bareCombineAddin
remotes::install_github("arunmhh/combinestring")
```
